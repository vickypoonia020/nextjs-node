# nextjs-node
 Steps for setup:-

Client- 
 Initials:- 
 1. git init(To get node based server setup)
 2. install Express
 3. touch server.js    -this command will create a server.js file

    _After Clone
 1. Have the latest Node version
 2. npm i  / npm i -f
 3. npm run dev  -To run nextjs app 



Server-
 Initials:- 
 1. git init(To get node based server setup)
 2. install Express
 3. touch server.js    -this command will create a server.js file

    _After Clone
 1. npm i  / npm i -f
 2. npm start   -To start the server
